# DinoRunnerNEAT
AI learns to play Dino Runner based on the offline Google Chrome game using NEAT.

NEAT part is based NEAT-Python library example and this <a href ="https://www.youtube.com/watch?v=MMxFDaIOHsE">video</a>.
